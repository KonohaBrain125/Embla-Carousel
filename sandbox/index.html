Features

Embla is a lightweight and simple carousel plugin for the web. Its purpose is to provide simple and extensible carousels that feels natural to interact with. It's 100% open source and free to use on both personal and commercial projects.

  • Loop
  • Align
  • Speed
  • Draggable
  • StartIndex

Loop

Determines if the carousel should loop when start or end is reached. Default is false.

{ loop: false }
{ loop: true }

Align

Aligns the slides relative to the carousel viewport. Default is 'center'.

{ align: 'center' }
{ align: 'start' }
{ align: 'end' }

Speed

Carousel speed when using buttons to navigate. A higher number will make transitions faster. Pointer events are not affected by this. Default is 10.

{ speed: 10 }
{ speed: 20 }

Draggable

Allow mouse and touch interactions to move the carousel. Default is true.

{ draggable: true }
{ draggable: false }

Startindex

Zero based index of the starting slide when carousel mounts. Default is 0.

{ startIndex: 0 }
{ startIndex: 2 }
